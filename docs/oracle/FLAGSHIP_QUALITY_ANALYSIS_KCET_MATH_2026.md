## 🔧 ROOT CAUSE ANALYSIS

### Why Generated Questions Miss KCET Style

**1. AI Prompt Over-emphasizes "Competitive Rigor"**
```typescript
"MISSION: PRE-EMPTIVE COMPETITIVE FORECAST"
"ULTIMATE-RIGOR MCQ questions"
"Create 1 question that is a 'New Twist' never seen before"
```
**Result:** AI generates JEE-style questions, not KCET-style

**2. Missing KCET-Specific Constraints**
- No mandate for word problems
- No pattern recognition emphasis
- No "practical context" requirement
- No "simple numerical values" constraint

**3. Oracle Mode Directives Too Abstract**
```typescript
directives: [
  "Heuristic Shortcut Mapping",
  "Property-based matrix mechanics",
  "TRAP: Formula-Baiting in Calculus"
]
```
**Result:** Produces clever/tricky questions, not KCET's straightforward style

**4. Identity Bank Focus**
- System tries to use high-confidence identities
- But identities are patterns, not question templates
- KCET questions follow templates (word problem, numerical, pattern)
