## ⚡ Quick Reference

### What's Stored Where?

| Parameter | Primary Storage | Backup/Reference |
|-----------|----------------|------------------|
| **IDS Target** | `ai_universal_calibration.intent_signature.idsTarget` | `rei_evolution_configs.ids_baseline` |
| **Rigor Velocity** | `ai_universal_calibration.rigor_velocity` | `rei_evolution_configs.rigor_drift_multiplier` |
| **Synthesis** | `ai_universal_calibration.intent_signature.synthesis` | `rei_evolution_configs.synthesis_weight` |
| **Difficulty %** | `ai_universal_calibration.intent_signature.difficultyProfile` | `exam_historical_patterns.difficulty_*_pct` |
| **Question Types** | `ai_universal_calibration.intent_signature.questionTypeProfile` ✨ | `QUESTION_TYPE_ANALYSIS_2021_2025.json` |
| **Identity Confidence** | `kcet_math.json.identities[].confidence` | - |
| **Board Signature** | `ai_universal_calibration.board_signature` | - |

### Priority Order (REI Engine)

1. **`ai_universal_calibration`** (MASTER OVERRIDE) ← Always used if exists
2. **`rei_evolution_configs`** (Fallback)
3. **`exam_historical_patterns`** (Historical trends)
4. **Baseline defaults** (Last resort)
