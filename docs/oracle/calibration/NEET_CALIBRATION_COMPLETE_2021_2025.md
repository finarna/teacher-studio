## Calibration Methodology

### Confidence Weight Calculation

Based on empirical frequency (avg questions per year):

| Avg/Year | Confidence | Category | Meaning |
|----------|------------|----------|---------|
| ≥ 3.0 | 2.0× | High | Appears 3+ times/year |
| 2.0-2.9 | 1.5× | Medium-High | Appears 2-3 times/year |
| 1.0-1.9 | 1.0× | Medium | Appears 1-2 times/year |
| 0.5-0.9 | 0.7× | Low | Appears occasionally |
| < 0.5 | 0.2× | Very Low | Rare appearance |

### How Calibration Works in AI Generator

```typescript
// Example: Chemistry question generation
const rawPrediction = AI.predict(); // Returns equal distribution

// Apply empirical calibration
const calibrated = rawPrediction.map(topic => ({
  ...topic,
  expectedQuestions: topic.expectedQuestions × confidence[topic.id]
}));

// Re-normalize to maintain total count
const total = calibrated.reduce((sum, t) => sum + t.expectedQuestions, 0);
const final = calibrated.map(t => ({
  ...t,
  expectedQuestions: (t.expectedQuestions / total) * 50
}));
```

**Result:** High-frequency topics get more questions, low-frequency topics get fewer, matching actual exam patterns.
