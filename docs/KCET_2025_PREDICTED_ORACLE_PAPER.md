**Question:** If $A = \{x : x \text{ is a multiple of 3}, x < 100\}$ and $B = \{x : x \text{ is a multiple of 5}, x < 100\}$, the number of elements in the symmetric difference $A \Delta B$ is:
Options: (A) 40 (B) 39 (C) 46 (D) 33
**Correct:** (B) 39
**Machine Mode Logic:** 2025 shifts from simple union/intersection to Symmetric Difference ($A \Delta B$) to test set-theoretic subtraction speed. $n(A)=33, n(B)=19, n(A \cap B)=6$. Result: $(33-6) + (19-6) = 39$.

**Q18 [Matrices / High]**
**Question:** If $A$ is a $3 \times 3$ matrix such that $|adj(A)| = 64$, then $|A|$ is:

**Q40 [Calculus / IDS Directive 2]**
**Question:** Evaluate $\int e^x \left( \frac{1 + \sin x \cos x}{\cos^2 x} \right) dx$.
Options: (A) $e^x \tan x + C$ (B) $e^x \sec x + C$ (C) $e^x (\tan x + 1) + C$ (D) $e^x \sin x + C$
**Correct:** (A) $e^x \tan x + C$
**Machine Mode Logic:** **IDS TWIST.** The integrand is $e^x(\sec^2 x + \tan x)$. This follows $e^x(f(x) + f'(x))$ but requires the student to decompose the fraction first. 2025 adds the "trig-split" layer.

**Q45 [Differential Equations / High]**
**Question:** The integrating factor of the differential equation $x \frac{dy}{dx} - y = 2x^2$ is:

**Correct:** (C) $1/x$
**Machine Mode Logic:** Shift from solving the full equation to identifying the IF (Integrating Factor) to save time.

**Q52 [3D & Vectors / IDS Directive 3]**
**Question:** Find the shortest distance from the point $(1, 2, 3)$ to the plane $\vec{r} = \lambda(\hat{i} + \hat{j}) + \mu(\hat{j} + \hat{k})$.
Options: (A) $2/\sqrt{3}$ (B) $1/\sqrt{3}$ (C) 0 (D) $\sqrt{6}$
**Correct:** (B) $1/\sqrt{3}$
**Machine Mode Logic:** **IDS TWIST.** The plane is given as a linear combination of two vectors. Students must first find the normal vector $\vec{n} = (\hat{i}+\hat{j}) \times (\hat{j}+\hat{k}) = \hat{i} - \hat{j} + \hat{k}$ before applying the distance formula. This is the "Synthesis Peak" merging Vector products with 3D geometry.
