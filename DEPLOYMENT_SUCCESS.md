## 🆘 Troubleshooting

### If Blog URLs Show 404:
- Clear browser cache and retry
- Check Vercel deployment status
- Verify dist/blog/ folder was deployed

### If Meta Tags Missing:
- View page source (Ctrl+U)
- Look for `<meta>` tags in `<head>`
- If missing, rebuild and redeploy

### If Social Preview Broken:
- Use Facebook Debug "Scrape Again"
- Wait 24 hours for cache to update
- Verify OG image URL is absolute

### If No Google Impressions After 1 Week:
- Check indexing status in GSC
- Request indexing again
- Verify robots.txt allows crawling
- Ensure sitemap submitted correctly
